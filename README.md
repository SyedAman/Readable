# Readable

A social media application made using React and Redux. Users can create and edit posts as well as comment on posts and edit/delete them.

## How to run

### NPM

```sh
$ npm install
$ npm start
```

### Yarn

```sh
$ yarn
$ yarn start
```

### Docker

```sh
$ docker run -p 3000:3000 syednashikaman/readable:latest
```
